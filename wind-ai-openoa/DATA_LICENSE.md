# Data source and license

The files downloaded by `scripts/download_data.py` are the example data shipped
by the OpenOA repository for the ENGIE "La Haute Borne" wind farm.

Source repository:
https://github.com/NatLabRockies/OpenOA/tree/main/examples/data

Original ENGIE dataset identifier recorded by OpenOA:
`d543716b-368d-4c53-8fb1-55addbe8d3ad`

The OpenOA example data directory states that the data are based on ENGIE data
published under France's Open Licence version 2.0.
